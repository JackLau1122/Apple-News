# Apple-New快捷指令使用教程

请前往我的机器人或者Github发Issues反馈使用成功率以便我加以改进。

⚠️ 须知:该过程会将你的系统地图从高德地图切换成国外的TOMTOM地图。(使用蜂窝数据打开地图即可还原)  
#. 请设置手机地区与订阅 Apple news地区一致  
#. 脚本运行完成后关闭系统自带地图。并且不要在使用蜂窝数据时打开系统自带地图，否则立马失效。  
#. 请在底部查看，并按照图片内容设置定位选项。


💁🏻‍♂️请参考神级规则设置Apple News的分流(全局代理也行)  
❤️如需更加便捷使用，可点击此界面右上角"..."将本快捷指令添加到主屏幕。
